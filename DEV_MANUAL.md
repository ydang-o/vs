# 合伙人投票系统 - 微信小程序开发手册

## 1. 项目概述
本项目为“合伙人投票系统”的微信小程序端，基于 **uni-app + Vue 3** 开发。
系统聚焦于合伙人投票决策，提供轻量化的议案查看、投票（含委托）、结果查询等功能。

## 2. 技术栈
- **框架**: uni-app (Vue 3)
- **开发工具**: HBuilderX
- **UI风格**: 扁平化设计，主色调 #3B82F6

## 3. 目录结构
项目采用 HBuilderX 默认的非 CLI 结构，方便直接拖入运行。
```
miniprogram/
├── pages/
│   ├── login/      # 登录页
│   ├── index/      # 首页（投票列表）
│   ├── detail/     # 议案详情页（含投票逻辑）
│   ├── result/     # 结果展示页
│   └── profile/    # 个人中心
├── static/         # 静态资源（图标、图片）
├── App.vue         # 应用入口
├── main.js         # 入口文件
├── pages.json      # 页面路由及TabBar配置
├── manifest.json   # 应用配置
├── uni.scss        # 全局样式
└── index.html      # H5入口（HBuilderX自动维护）
```

## 4. 快速开始 (HBuilderX)

### 4.1 导入项目
1. 打开 **HBuilderX**。
2. 点击菜单栏 **文件** -> **打开目录**。
3. 选择本项目中的 `miniprogram` 文件夹。

### 4.2 运行到微信小程序
1. 在 HBuilderX 中打开项目后，确保您已安装微信开发者工具并登录。
2. 在 HBuilderX 顶部菜单栏点击 **运行** -> **运行到小程序模拟器** -> **微信开发者工具**。
3. HBuilderX 将会自动编译并启动微信开发者工具预览项目。

**注意**：
- 如果是首次运行，HBuilderX 可能会提示配置微信开发者工具的安装路径。
- 请确保微信开发者工具已开启“服务端口”（设置 -> 安全设置 -> 服务端口：开启）。

## 5. 功能模块实现

### 5.1 登录 (pages/login)
- 模拟微信一键登录流程。
- 登录成功后跳转至首页。

### 5.2 投票列表 (pages/index)
- 顶部 Tab 切换“待投票”和“已结束”。
- 列表卡片展示议案编号、标题、状态、截止时间。
- 紧急任务（<24小时）显示红色标记。

### 5.3 议案详情与投票 (pages/detail)
- 展示议案详情（支持富文本）。
- 底部操作栏：
  - **本人投票**: 弹窗选择同意/反对/弃权。
  - **委托投票**: 若有委托权限，显示委托人姓名及投票按钮。
  - **查看结果**: 已投票或已结束时展示。

### 5.4 结果展示 (pages/result)
- 汇总展示同意/反对/弃权的票数及占比（进度条）。
- 列表展示详细投票记录（实名模式）。

### 5.5 个人中心 (pages/profile)
- 展示合伙人基本信息（姓名、层级、出资比例）。
- 退出登录功能。

## 6. UI设计规范
- **主色**: #3B82F6 (蓝色)
- **卡片**: 白色背景，2px 浅灰边框，无阴影。
- **字体**: 标准系统字体，重要信息加粗。
